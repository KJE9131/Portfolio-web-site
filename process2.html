Overview

경상남도 산청군의 관공서 홈페이지 산청군청은 화이트와 블루계열의 색상을 위주로 제작된 페이지입니다.
내용은 많지만 디자인이 심플하고 레이아웃의 정리가 깔끔해서 선택하게되었습니다.
첫페이지의 슬라이드 배너만 3가지 종류로 제작이 어렵더라도 배울점이 많아 제작하게되었습니다.

Web Publishing

Semantic Markup

웹 표준을 바탕으로 꼼꼼한 'Semantic Markup'을 작성합니다.

Movement

'패밀리사이트' 와 'language' 등 버튼 효과와
슬라이드 배너 등 다양한 동작을 JQuery로 처리했습니다.

반응형 Layout

미디어쿼리를 사용하여 반응형 페이지를 제작했습니다.
class ‘pc’ 와 class ‘mob’를 사용하여 pc와 모바일의 변화를 쉽게 제작했습니다.

Portfolio

Epilogue

산청군청 홈페이지는 모든 이론, 실습 수업이 끝난 뒤 처음으로 진행한 홈페이지입니다.

처음 산청군청을 선택했을때 막연히 많은 기술과 예쁜 디자인을 하고싶어서 선택했지만 막상 제작하다보니 벽에 부딪힐 때가 많았습니다.
많은 시행착오가 있었지만, 주변 동료와 선생님의 도움을 받으며 끝까지 제작할 수 있었습니다.
도움 받은것에서 끝내지 않고 메모도하고 제 스타일로 다시한번 정리해 보는 등 제것으로 만들기 위해 노력했습니다.

자주사용하는 스크립트나 class명을 찾아보아 메모하여 저만의 노하우를 쌓을 수 있었던 계기가 된 것 같아 무척 뜻깊은 시간이었습니다.
