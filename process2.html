Overview

경상남도 산청군의 관공서 홈페이지로 화이트와 블루 계열의 색상을 위주로 제작된 페이지입니다.
많은 정보성 텍스트와 디자인이 심플하고 레이아웃의 정리가 깔끔해서 선택하게 되었습니다.
메인 페이지의 슬라이드 배너만 3가지 종류로 시간이 소요되더라도 배울 점이 많아 보여 제작하게 되었습니다.

Web Publishing

Semantic Markup

'Semantic Markup'을 준수하여 제작하고 W3C의 검증을 받았습니다.

Movement

'패밀리사이트' 와 'language' 등 버튼 효과와
슬라이드 배너 등 다양한 동작을 JQuery로 구현했습니다.

반응형 Layout

미디어쿼리를 사용하여 반응형 페이지를 제작했습니다.
class ‘pc’ 와 class ‘mob’를 사용하여 pc와 모바일의 변화를 간단히 제작했습니다.

Portfolio

Epilogue

산청군청 홈페이지는 훈련과정 중 이론, 실습수업이 끝난 뒤 첫 프로젝트입니다.

퍼블리싱에 대한 욕심으로 많은 기술과 예쁜 디자인을 하고 싶어서 선택했지만 제작 당시 벽에 부딪힐 때가 많았습니다.
시행착오는 있었지만, 구글링하며 직접 찾아보고 선생님께 도움도 받으며 끝까지 제작할 수 있었고 새로운 기술을 배울 수 있었던 시간이었습니다.
도움받은 부분은 다시 한번 정리해 보며 제 것으로 만들기 위해 노력했습니다.
